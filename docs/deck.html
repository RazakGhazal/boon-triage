09The engineer's verdictboon●academy

The ceiling isn't the model. It's the data.

Sparse, abstract data lets even perfect logic see each student only from a kilometre up. The real job isn't a better classifier — it's governing the data the model is built on.

10Govern the inputboon●academy

The notes are sparse —
and uneven by luck.

62% of students have no note at all — 30% of failers invisible.
Coverage swings 21% → 55%; the largest classroom is least documented.
One facilitator carries up to 38 students — no time to note each.
Govern the process, not the output
1
Keep the free text. That mess is what the LLM tolerates — no rigid forms.
2
Add one structured field — “did it land?” — proxy becomes ground truth.
3
Fix the ratio. Coverage is a staffing problem before a model one.
